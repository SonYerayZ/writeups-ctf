# 🚩 Writeups CTF 

Colección de writeups de los CTFs en los que he participado. Aquí documento el proceso de resolución de retos, herramientas utilizadas y conceptos aprendidos.

## 🗂️ Categorías

- **Web** – Inyecciones, XSS, SSRF, autenticación rota...
- **Pwn** – Buffer overflows, exploits de binarios...
- **Reversing** – Ingeniería inversa de binarios y aplicaciones
- **Crypto** – Criptografía clásica y moderna
- **Forensics** – Análisis de ficheros, memoria, redes...
- **Misc** – Todo lo demás

## 🛠️ Herramientas habituales

`pwntools` · `Ghidra` · `GDB` · `Burp Suite` · `Wireshark` · `CyberChef` · `John the Ripper` · `binwalk`
